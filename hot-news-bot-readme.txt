# Hot News Bot

Hot News Bot - это автоматизированный Telegram бот, специализирующийся на сборе, обработке и публикации горячих новостей о знаменитостях и событиях шоу-бизнеса.

## Описание

Бот выполняет следующие функции:
- Сбор новостей из различных источников, специализирующихся на новостях о знаменитостях
- Анализ и оценка "горячести" и скандальности новостей
- Автоматическая публикация отобранных новостей в Telegram канале
- Отслеживание статистики публикаций и вовлеченности аудитории
- Определение оптимального времени для публикаций

## Основные особенности

- Фокус на скандальных и сенсационных новостях
- Категоризация новостей (например, "скандалы", "отношения", "звездные войны")
- Интеграция с календарем событий шоу-бизнеса (премьеры, церемонии награждения и т.д.)
- Автоматическое определение "горячих" новостей
- Разнообразие контента для поддержания интереса аудитории

## Установка

1. Клонируйте репозиторий:
   ```
   git clone https://github.com/your-username/hot-news-bot.git
   cd hot-news-bot
   ```

2. Создайте виртуальное окружение и активируйте его:
   ```
   python -m venv venv
   source venv/bin/activate  # Для Linux/Mac
   venv\Scripts\activate  # Для Windows
   ```

3. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

4. Создайте файл `.env` в корневой директории и добавьте следующие переменные:
   ```
   TELEGRAM_BOT_TOKEN=your_bot_token
   TELEGRAM_CHANNEL_ID=your_channel_id
   ADMIN_CHAT_ID=your_admin_chat_id
   ```

## Использование

1. Запустите бота:
   ```
   python main.py
   ```

2. Бот начнет собирать и публиковать горячие новости о знаменитостях автоматически.

3. Для управления ботом используйте следующие команды в чате с ботом:
   - /start - Начать работу с ботом
   - /help - Показать справку
   - /status - Проверить статус бота
   - /stats - Показать статистику публикаций
   - /top - Показать топ статей
   - /events - Показать предстоящие события в мире шоу-бизнеса
   - /pause - Приостановить публикации
   - /resume - Возобновить публикации
   - /scores - Показать таблицу оценок статей
   - /optimal_time - Показать оптимальное время публикаций

## Тестирование
